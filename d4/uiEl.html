form태그 추가
    fieldset 태그란
  • 폼 요소들을 그룹화할 때 사용
  • 시각적으로 박스 형태로 묶어주며, 접근성 향상에도 도움
  • 예: 이름, 연락처, 주소 등 관련된 입력 필드를 하나의 그룹으로 묶을 때
    legend 태그란?
  • fieldset의 첫 번째 자식 요소로 사용
  • 그룹의 제목 또는 설명 역할
  • 스크린 리더 사용자에게 그룹의 목적을 알려주는 데 중요

label 태그 안에 input을 추가할 경우,
Input 입력 칸 클릭을 하지 않아도 label 태그 안에 Text를 클릭해도 입력칸이 활성화되어짐
label 태그 안에 있는 for=""은 input 태그 안의 id 값과 일치해야됨

type:radio → O 모양의 체크칸

legend 태그는 이와 같이 표 전체의 제목 설정에 용이함
