자식 선택자 → 부모 태그 `''>'' 자식
ex) ul > li{(css 스타일링)}


후손 선택자 → 부모 태그 "(공백) " 자식
ex)ul li{(css 스타일링)}


가상 선택자 → 태그 ":" ~
ex) container:hover { color: white; background-color: blueviolet; }
=> container class name 뒤에 (가상 선택자) : 를 붙인 후 hover css 스타일 선택
