선택자 사용
form태그 추가
    구조 가상 선택자란?
  • CSS에서 HTML 요소의 구조나 위치에 따라 스타일을 적용할 수 있게 해주는 선택자
  • 여러 요소가 나열되있을 경우
    class 및 id를 사용하지 않고도 스타일을 지정할 요소가 몇번째인지를 지정하여 스타일 적용이 가능함
    부모태그: 구조가상선택자{}
  • first-child : 부모 요소의 첫 번째 자식 요소에 적용. 예) .ul li:nth-child(1) {color: blue;}
  • last-child : 부모 요소의 마지막 자식 요소에 적용
  • nth-child(n) : 부모 요소의 n번째 자식 요소에 적용
  • nth-last-child(n) : 뒤에서부터 부모 요소의 n번째 자식 요소에 적용
  • only-child : 부모 안에 자식이 하나뿐일 때 적용
  • first-of-type : 같은 타입 중 첫 번째 요소에 적용
  • last-of-type : 같은 타입 중 마지막 요소에 적용
  • nth-of-type(n) : 같은 타입 중 n번째 요소에 적용
  • nth-last-of-type(n) : 같은 타입 중 뒤에서 n번째 요소에 적용
  • only of type : 같은 타입이 하나뿐일 때 적용
